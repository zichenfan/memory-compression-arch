# Arxiv AI硬件加速论文日报系统 - 部署指南

## 📋 概述

Arxiv AI硬件加速论文日报系统是一个自动化工具，每天从Arxiv抓取最新的AI硬件加速相关论文，筛选高质量内容，生成结构化日报，并自动推送到飞书。

## 🚀 快速开始

### 1. 环境要求
- Python 3.10+
- Git
- 飞书开发者账号

### 2. 克隆仓库
```bash
git clone https://github.com/zichenfan/openclaw-jason-workspace.git
cd openclaw-jason-workspace/projects/arxiv-ai-hardware-daily-digest
```

### 3. 安装依赖
```bash
pip install -r requirements.txt
```
